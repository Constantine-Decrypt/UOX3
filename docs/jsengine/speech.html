API and Event handling

Speech Related Functions

SysMessage
Prototype
SysMessage( UOXSOCKET toSendTo, string toSend );
PurposeSends a string as a sysmessage to toSendTo
Example of Usage
SysMessage( 0, "Isn't the world great?" );

 

TextMessage
Prototype
TextMessage( int srcChar, string message );
PurposeCauses srcChar to say message.
Example of Usage
TextMessage( srcChar, "Shiver me timbers!" );

 

EmoteMessage
Prototype
EmoteMessage( int srcChar, string message );
PurposeCauses srcChar to emote message. Note that you need to manually put in the start and end *
Example of Usage
EmoteMessage( srcChar, "Shiver me timbers!" );

 

BroadcastMessage
Prototype
BroadcastMessage( string toSend );
PurposeSends toSend to every client in the world
Example of Usage
BroadcastMessage( "You're all going to die a hideous and painful death" );

 

ConsoleMessage
Prototype
ConsoleMessage( string toSend );
PurposePrints out a string to the server console
Example of Usage
ConsoleMessage( "Isn't the world great?" );

 

YellMessage
Prototype
YellMessage( int srcChar, string message );
PurposeCauses srcChar to yell message.
Example of Usage
YellMessage( srcChar, "Shiver me timbers!" );

 

WhisperMessage
Prototype
WhisperMessage( int srcChar, string message );
PurposeCauses srcChar to whisper message.
Example of Usage
WhisperMessage( srcChar, "Shiver me timbers!" );

 

SubStringSearch
Prototype
bool SubStringSearch( strToSearch, searchString );
PurposeReturns true if the string searchString is to be found inside strToSearch. Case insensitive search.
Example of Usage
var bFound = SubStringSearch( strSaid, "HELP" );
